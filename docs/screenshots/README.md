# Screenshots

Drop PNG screenshots into this folder using the filenames below. The main README.md references these paths.

| # | Filename | What to capture |
|---|----------|-----------------|
| 1 | `dashboard.png` | Dashboard page showing stat cards, priority bar chart, MITRE donut chart, and recent alerts table |
| 2 | `alerts-feed.png` | Alerts feed page with a search term entered and results filtered |
| 3 | `alert-detail.png` | Alert detail page for the Cobalt Strike beacon alert (ALERT-2026-0455) showing confidence bar, MITRE mapping, and recommended actions |
| 4 | `reports.png` | Reports page showing priority distribution pie chart, MITRE tactics bar chart, and techniques breakdown |
| 5 | `settings.png` | Settings page with integration toggles and triage tuning sliders |
| 6 | `terminal-triage.png` | Terminal output from running `python main.py` showing triage reports being printed |
| 7 | `slack-notification.png` | Slack channel showing a triage notification with priority, summary, and MITRE mapping |
| 8 | `jira-ticket.png` | Jira issue created for a CRITICAL/HIGH escalated alert |

## Tips

- Use **1280x800** or higher resolution for crisp rendering on GitHub.
- Dark mode screenshots match the dashboard theme.
- Crop browser chrome if possible — just the page content.
- For terminal screenshots, use a dark terminal theme for consistency.
